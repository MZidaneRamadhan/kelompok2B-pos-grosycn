# kelompok2B-pos-grosycn
Point of Sale (POS) 

POS System adalah aplikasi Point of Sale berbasis desktop yang dirancang untuk membantu pelaku usaha dalam mengelola transaksi penjualan secara efisien. Aplikasi ini dikembangkan sebagai proyek tugas besar mata kuliah Proyek 1.

Sistem ini mencakup fitur manajemen produk, transaksi kasir, laporan penjualan, serta manajemen pengguna dengan sistem autentikasi.
